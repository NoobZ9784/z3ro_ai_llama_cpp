# z3ro_ai_llama_cpp
Z3RO AI is an local integration of local LLM models of GGUF file format and typescript based Node Js environment.

To work with this application you can download any LLM Model you want from hugging face based on the system configuration. Create a 'Models' folder and mention your .gguf file into index.ts(25).

Note: If you have GPU in your system set gpu to 'vulkan' (recomended) or 'cuda' and set context size based on VRAM or RAM.
